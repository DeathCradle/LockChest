Olympus MineCraft - LockChest
===========

A chest protection tool, Allowing Locking & Picking!

Features
-----------
	* Configurable Items for picks
	* Chest protection
	* Picking (configurable)
	* Configurable Success Rates
	* Multi World
	* Permissions - With or Without
